Foodies

The Foodies website was a collaborative web development project I worked on during my Code First Girls class, alongside three other teammates from Canada, Italy, and Pakistan. The website aimed to celebrate food culture fusion by showcasing diverse culinary traditions from each of our backgrounds. Me and my team built the website using HTML, CSS, and JavaScript, ensuring a visually appealing and interactive user experience. Each member was responsible for designing and developing a unique page, incorporating elements like recipes, food history, cultural significance, and interactive features such as image galleries, hover effects, and buttons. This project was particularly significant because it was my first collaborative web development experience, allowing me to practice teamwork, version control, and problem-solving while applying my growing frontend development skills.
